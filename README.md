Rust was created to solve common problems found in low-level languages like C and C++.

### Why Rust?

- **Memory Safety**  
  Prevents null pointer dereference, buffer overflow, and use-after-free at compile time.

- **High Performance**  
  Comparable to C/C++ with zero-cost abstractions.

- **No Garbage Collector**  
  Predictable performance and low-level control.

- **Concurrency Without Data Races**  
  Safe multi-threading enforced by the compiler.

- **Modern Tooling**  
  Comes with an excellent package manager and build system.
